# RoomEscape

## 게임방법

```
1. room1에서 room2 이동합니다.
2. 키 찾기 버튼을 누르면 키의 위치가 랜덤으로 표시됩니다.
3. 그러나 랜덤으로 위치하는 키를 클릭하는 것이 아니라 '키 찾기' 버튼의 키를
   클릭해야 얻을 수 있습니다.
4. 키를 얻으면 room1을 통해 room3로 이동합니다.
5. board를 클릭하여 게임의 룰을 확인합니다.
6. 음표 버튼을 클릭하여 생성되는 오디어 플레이어로 음악을 재생합니다.
7. 문제에 해당하는 답은 기타와 마림바입니다.
8. 기타와 마림바 버튼을 클릭하면 보드 위에 숨어있던 탈출구가 나오며 탈출이 가능합니다.
```
